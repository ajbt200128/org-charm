* Org Syntax Showcase

** Text Formatting

Org mode supports various inline formatting:

- *Bold text* for emphasis
- /Italic text/ for subtle emphasis
- _Underlined text_ for highlighting
- +Strikethrough+ for deleted content
- =Verbatim= for literal text
- ~Code~ for inline code

** Lists

*** Unordered Lists

- First item
  - Nested item A
  - Nested item B
    - Deeply nested
- Second item
- Third item

*** Ordered Lists

1. First step
2. Second step
3. Third step
   1. Sub-step A
   2. Sub-step B

*** Checklists

- [X] Complete task
- [-] Partially complete
- [ ] Pending task

** Code Blocks

Here's some Go code with syntax highlighting:

#+BEGIN_SRC go
package main

import (
    "fmt"
    "github.com/charmbracelet/lipgloss"
)

func main() {
    style := lipgloss.NewStyle().
        Bold(true).
        Foreground(lipgloss.Color("#7aa2f7"))

    fmt.Println(style.Render("Hello, Org-Charm!"))
}
#+END_SRC

And some Python:

#+BEGIN_SRC python
def fibonacci(n: int) -> list[int]:
    """Generate Fibonacci sequence."""
    fib = [0, 1]
    for i in range(2, n):
        fib.append(fib[i-1] + fib[i-2])
    return fib[:n]

print(fibonacci(10))
#+END_SRC

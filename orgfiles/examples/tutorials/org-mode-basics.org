* What is Org-Mode?

Org-mode is a powerful document editing and organization system
for Emacs[fn:1]. It supports outlines, notes, planning, and more.

* Basic Syntax

** Headings

Use asterisks for headings:
- =*= for level 1
- =**= for level 2
- =***= for level 3

** Text Formatting

- *bold* text uses asterisks
- /italic/ text uses slashes
- _underline_ uses underscores
- =verbatim= uses equals
- ~code~ uses tildes
- +strikethrough+ uses plus

** Lists

Unordered lists use =-= or =+=:
- Item one
- Item two
  - Nested item
- Item three

Ordered lists use numbers:
1. First item
2. Second item
3. Third item

** Links

Links use double brackets: =[[url][description]]=

Example: [[https://orgmode.org][Org-Mode Website]]

* Footnotes in Org-Mode

Footnotes are created with =[fn:name]=[fn:2]. They can be inline
or defined separately.

Inline footnotes look like this[fn:inline:This is an inline footnote
that contains the definition right here in the text.].

* Code Blocks

Source code blocks support syntax highlighting[fn:code]:

    print("Hello, Org-Mode!")
